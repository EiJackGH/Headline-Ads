How to integrate programmatically?

Besides configuring attributes on your script element, you can call our JavaScript SDK API directly.

A. Declarative attributes

Add data attributes to the script. The script automatically reads your custom setup and spawns the banner without you writing any custom script tags or initialisation hooks. 

<script
  src="headline-ads.js"
  data-id="header-bar"
  data-text="Join us on Slack!"
  data-link="https://slack.com"
  data-position="top"></script>

B. Imperative JavaScript API

Trigger, change or dismiss banners inside single-page applications (React, Vue, Svelte, Angular) by accessing the global namespace window.HeadlineAds object. 

HeadlineAds.init({
  id: 'spa-announcement',
  text: 'SPA Custom message!',
  bgColor: '#10b981',
  position: 'inline',
  target: 'target-node'
});
